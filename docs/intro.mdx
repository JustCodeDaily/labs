# Introduction

Welcome to the **Interview Prep** guide. This resource is designed to help you master technical interviews, covering JavaScript, React, Machine Coding, and Data Structures & Algorithms (DSA).

## How to use this guide

Choose a category from the sidebar to jump into specific topics. Each category is divided into three skill levels:
- **Beginner**: Foundational concepts.
- **Intermediate**: Modern practices and intermediate problem solving.
- **Expert**: Advanced architecture and deep-dive internals.
