# JavaScript Beginner

Master variables, data types, and flow control.

```js live
// This becomes an editable + runnable code block
function App() {
  const [count, setCount] = React.useState(0);
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}
```
